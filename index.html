HTML

Elements
Tag Name Description
<body> Body Defines your page's body
<div> Division Defines a division or a section in your page
<h1> Biggest Heading Use headings to add hierarchy to your text
<h6> Smallest Heading Use headings to add hierarchy to your text
<p> Paragraph Paragraph represents normal text
<span> Span Use Span to mark-up a part of a text
<strong> Bold text Renders your text bold
<br> Line break Insert single line breaks in a text (no closing tag)
<ol> Ordered list Defines an ordered list. An ordered list can be numerical or alphabetical
<ul> Unordered list Defines an unordered (bulleted) list
<li> List item Defines a list item
<img src=""> Image Embed an image in your page (no closing tag)
<img src="" alt=""> Alt text Specifies an alternate text for the image, if the image for some reason cannot be displayed
<video src=""> Video Embed a video in your page (text between opening and closing tag in case browser doesn't support video
<audio><source src=""> Audio Is used to embed sound content in a document
Structure
Tag Name Description
<!DOCTYPE html> Document type All HTML documents must start with a <!DOCTYPE> declaration
<html> HTML Represents the root of your HTML document.
<head> Head The head is a container for metadata like title, styles, scripts etc.
<title> Page title Defines the page title, and shown in browser tab.
<!-- --> Comments The comment tag is used to insert comments in the source code. Comments are not displayed in the browsers.
<a href=""> Anchor link Defines a hyperlink, which is used to link from one page to another.
<a href=""><img src=""> Image link Use it to use an image as a link
<a href="#sectionID"> Link to same page Links to another section on the same page
<a href="./page.html"> Link to local file Links to a local file saved in the same folder/repository
target="_blank" Target attribute Specifies where to open the linked document (example: new tab)
Tables
Tag Name Description
<table> Table Defines a HTML table
<thead> Table head Is used to group header content in an HTML table
<tbody> Table body Is used to group the body content in an HTML table
<tfoot> Table footer Is used to group the footer content in an HTML table
<th> Table heading Defines a header cell in an HTML table
<tr> Table row Defines a row in an HTML table
<td> Table data Defines a standard data cell in an HTML table
<rowspan="3"> Rowspan attribute Specifies the number of rows a cell should span
<colspan="3"> Colspan attribute Specifies the number of columns a cell should span
Semantic HTML
Tag Name Description
<header> Header Represents a container for introductory content or a set of navigational links
<nav> Navigation Defines a set of navigation links
<main> Main Specifies the main content of a document
<footer> Footer Defines a footer for a document or section
<section> Section Defines a section in a document
<article> Article Specifies independent, self-contained content

<figure> Figure Specifies self-contained content, like illustrations, diagrams, photos, code listings, etc.

CSS

Selectors
Tag Name Description
<link rel="stylesheet" href="styles.css"> Link element Defines the relationship between the current document and an external resource. (no closing tag)
<p style="color: blue;"> Inline CSS Adds CSS style directly into the HTML document
.classname{} Class Selects elements with a specific class attribute
#idname{} ID Selects elements with a specific ID attribute
*{} All Selects all elements
p:hover{} State Provides different states for the same element
!important{} Important rule Is used on declarations to override any other declarations
element1, element2{} Group selectors To style several elements with the same style, separate each element name with a comma
element1.class1{} Chain selectors Selects the element described which also have the class described
.class1.class2{} Chain selectors Selects all elements with both class1 and class2 set within its class attribute
element1 element2{} Descendant selectors Selects element2 inside element2
Box Model
Property Name Description
Content The content of the box, where text and images appear
Padding Clears an area around the content. The padding is transparent
Border A border that goes around the padding and content
Margin Clears an area outside the border. The margin is transparent
width: 50px Width Sets the width of an element's content
height: 50px Height Sets the height of an element's content
border: 3px solid blue Border Defines the border appearance
border-radius: 5px Border radius Defines the radius of the element's corners.
padding: 5px Padding Defines the space between the content and its border
margin: 5px Margin Defines the space outside of the border (value: auto automatically centers the content)
overflow: hidden Overflow Specifies what should happen if content overflows an element's box
visibility: hidden Overflow Specifies whether or not an element is visible
box-sizing: border-box Box sizing Defines how the width and height of an element are calculated: should they include padding and borders, or not
Display and Positioning
Property Name Description
position: Position Specifies the type of positioning method used for an element
position: static Static position Default value. Elements render in order, as they appear in the document flow
position: absolute Absolute position The element is positioned relative to its first positioned (not static) ancestor element
position: fixed Fixed position The element is positioned relative to the browser window
position: relative Relative position The element is positioned relative to its normal position, so "left:20px" adds 20 pixels to the element's LEFT position
position: sticky Sticky position The element is positioned based on the user's scroll position
display: Display Specifies the display behavior (the type of rendering box) of an element
display: inline Inline Display Displays an element as an inline element. Any height and width properties will have no effect
display: block Block Display Displays an element as a block element. It starts on a new line, and takes up the whole width
display: inline-block Inline-Block Display Displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values
float: left Float specifies whether an element should float to the left, right, or not at all
clear: left Clear Specifies what should happen with the element that is next to a floating element
z-index: 2 Z index Specifies how far back or how far forward an element will appear on a web page when it overlaps other elements
Colors
Property Name Description
color: blue Name Color Specifies the color of the element
background-color: blue Bkacground Color Specifies the background color of the element
color: #92a8d1 HEX Color Hexadecimal digits can represent 16 different values using 0-9 and a-f
color: rgba(255, 5, 32, 0.5) RGBA Color RGBA should be supplied with 3 values (0-255) + opacity (0-1): red, green and blue
color: hsla(156, 20%, 60%, 0.5) HSLA Color RGBA should be supplied with 3 values + opacity (0-1): hue (0-360), saturation and lightness
Typography
Property Name Description
font-size: 50px Font size Sets the size of a font
font-weight: bold Font weight Sets how thick or thin characters in text should be displayed
font-family: "Times New Roman", Arial, Serif Font family Specifies the font for an element. Can hold several font names as a "fallback" system
font-style: italic Font style Specifies the font style for a text
text-align: left Text alignment Specifies the horizontal alignment of text in an element
text-transform: uppercase Text transformation Controls the capitalization of text
letter-spacing: 2px Letter spacing Increases or decreases the space between characters in a text
word-spacing: 2px Word spacing Increases or decreases the space between words in a text
line-height: 2px Line height Specifies the height of a line
@font-face {font-family: 'myfont'; src: url('../fonts/myfont.ttf) Import local font Use @font-face to import a local font. You must define a name for it.
